# Digital Wardrobe – Landing Page

A design-system-driven landing page for a digital wardrobe app that helps users organize clothes, shoes, accessories, and outfits in one place.

## Tech Stack
- Next.js
- React
- Tailwind CSS

🚧 In progress — landing page preview complete.

## Goals

- Demonstrate accessible, scalable UI
